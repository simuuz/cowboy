# natsukashii
### _"Joyful nostalgia"_
</br>
Rewrite of my GameBoy emulator in C++17 with ImGui and OpenGL (GLFW)</br>
It's shipped with a copyright-free, open source bootrom written by Optix:</br>
https://github.com/Hacktix/Bootix</br>

**Buttons are:**
- Z -> B
- X -> A
- Arrow keys -> DPAD
- Enter -> Start
- Right shift -> Select
